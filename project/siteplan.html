Website Plan: Idaho Outdoor Activities

Site Name

Idaho Outdoor Explorer

This name was selected because its the focus of the website outdoor activities in Idaho, including hunting, fishing, and hiking.

Site Purpose

The purpose of this website is to provide a comprehensive guide for outdoor people in Idaho. It will include sections for hunting, fishing, and hiking, offering tips, regulations, trail guides, and gear recommendations.

Scenarios

  • Where are the best hunting locations for elk in Idaho?
  • What are the current fishing regulations for Idaho rivers?
  • Can I find beginner-friendly hiking trails in Idaho's national forests?

Color Schema

The following color scheme will be used across the website:

  • Background color: #e9f5db (Light green) - used as the primary background color for a calm, outdoor feel.
  • Heading color: #1b5e20 (Dark green) - used for headings and accent text to represent nature.
  • Accent color: #f0f4c3 (Light yellow) - used for wireframes and highlighted sections.

Typography

  • Font 1: Arial (sans-serif) - This will be the main font for body text, ensuring readability and simplicity.
  • Font 2: Georgia (serif) - This font will be used for headings to create contrast with the body text and give a more rugged, outdoorsy feel.

Wireframe

This is a basic wireframe of the home page layout for both desktop and mobile views.

Desktop View:

  • Header with site name and navigation bar (links to Hunting, Fishing, Hiking subpages)
  • Hero image showcasing Idaho wilderness
  • Main content area with introduction to Idaho outdoor activities
  • Three section columns for Hunting, Fishing, Hiking (with brief descriptions)
  • Footer with contact info, social media links, and copyright

Mobile View:

  • Collapsible navigation menu with links to subpages
  • Hero image scaled down
  • Main content displayed in a single-column format
  • Footer with contact info and links, in a vertical stack
